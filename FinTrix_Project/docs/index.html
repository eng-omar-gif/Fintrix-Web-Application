FinTrix — Frontend JavaScript Documentation

FinTrix is a personal finance management web application built with HTML, CSS, JavaScript (frontend) and Python Django (backend).

Modules

Module File Description
script-budget script-budget.js Budget page: progress bars, form validation, grid/list toggle, contextual menus
script-dashboard script-dashboard.js Dashboard: monthly line chart, category donut chart, sidebar/range controls
script-home script-home.js Landing page: smooth scroll, newsletter form, scroll-triggered animations
script-login script-login.js Login page: async auth, password toggle, error display
script-notifications script-notifications.js Notifications: search filter, mark-as-read, staggered animations
script-reports script-reports.js Reports: date range, bar chart, donut chart, transaction table, CSV/PDF/Excel export
scripts-goals scripts-goals.js Goals page: sidebar toggle, hover effects, completed-goal styling
script-signup script-signup.js Sign-up: real-time validation, password strength meter, async registration
theme theme.js Theme manager: light/dark toggle, localStorage persistence, system preference
transactions transactions.js Transactions: add/delete, filtering, pagination, category loading, CSV export

Tech Stack

  • Frontend: HTML5, CSS3, Vanilla JavaScript (ES2017+)
  • Backend: Python 3, Django
  • Charts: Canvas API (line chart, bar chart), SVG (donut chart)
  • Auth API: /api/auth/login/, /api/auth/register/
  • Transactions API: /api/, /api/add/, /api/{type}/{id}/, /api/export/
  • Reports API: /reports/api/summary/, /reports/export/{csv|pdf|excel}/
